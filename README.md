## Getting Started
1. Clone the repository

   ```sh
   git clone https://github.com/ammad-sajjad/calculator_App.git        
   ```
3. Install all the packages by running
   ```sh
   flutter pub get
   ```
5. Run the App
   ```dart
   flutter run
   ```
